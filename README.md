ConnectDot

Overview

ConnectDot is a full-stack web application designed to help students enhance their skills, assess their knowledge, and explore career opportunities through a centralized platform.

The project is developed using React for the frontend and Django for the backend.

---

Features

User Authentication

- User Registration
- User Login
- Secure Authentication System

Course Enrollment

- Browse available courses
- Enroll in courses
- Track learning activities

Assessment Module

- Attend assessments
- Score evaluation
- Performance tracking

Job Opportunities

- Job recommendation interface based on assessment performance
- Career-focused dashboard

AI Chat Interface (Prototype)

- Student project discussion interface
- AI-powered chat UI design

Technical Reels Section (Prototype)

- Technical content feed interface
- Reels-style learning experience UI
